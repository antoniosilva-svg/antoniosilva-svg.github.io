# ORZHOV Portfolio

Portefólio profissional bilingue de António José Abreu e Silva, construído com Astro e Tailwind CSS.

## Desenvolvimento local

```sh
pnpm install
pnpm dev
```

## Publicação

O workflow em `.github/workflows/deploy.yml` publica automaticamente cada alteração enviada para o ramo `main` no GitHub Pages.

Antes de ligar `orzhov.pt`, configure o domínio personalizado nas definições do repositório em **Settings → Pages** e siga os registos DNS indicados pelo GitHub. Só então adicione um ficheiro `public/CNAME` com `orzhov.pt`.

## Conteúdo a completar antes da publicação

- Foto profissional e dados de contacto;
- CV, certificados e documentos autorizados para partilha;
- Datas, funções e descrição validada de cada experiência;
- Tradução completa da versão inglesa.
